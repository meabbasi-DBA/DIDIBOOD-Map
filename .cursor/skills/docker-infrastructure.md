# Docker Infrastructure Skill

## Objectives
- Containerize applications with production-grade images
- Standardize local and CI runtime environments
- Optimize image size, security, and build speed
- Support reproducible deployments across environments

## Responsibilities
- Write efficient Dockerfiles
- Define multi-stage builds
- Manage container networking and volumes
- Compose local development stacks
- Harden images for production use

## Deliverables
- Dockerfiles per service
- Docker Compose configurations
- Image build and tagging strategy
- Container security baseline
- Local development setup docs

## Standards
- Use minimal base images
- Run containers as non-root users
- Avoid unnecessary layers and secrets in images
- Pin dependency versions for reproducibility
- Keep build and runtime concerns separated
