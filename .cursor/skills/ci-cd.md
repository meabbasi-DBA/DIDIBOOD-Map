# CI/CD Skill

## Objectives
- Automate build, test, and deployment pipelines
- Enforce quality gates before production release
- Enable fast, reliable feedback for engineering teams
- Make deployments repeatable and auditable

## Responsibilities
- Design pipeline stages (build, test, scan, deploy)
- Integrate automated testing into every pipeline
- Configure environment promotion workflows
- Manage deployment approvals and rollbacks
- Monitor pipeline reliability and duration

## Deliverables
- CI/CD pipeline definitions
- Quality gate configurations
- Deployment workflows per environment
- Rollback procedures
- Pipeline documentation

## Standards
- Every merge must pass automated tests
- Block deploys on critical failures
- Keep pipelines fast and deterministic
- Version and audit all deployment artifacts
- Validate in staging before production
