# Kubernetes Skill

## Objectives
- Deploy and operate containerized workloads on Kubernetes
- Design resilient, scalable cluster configurations
- Enforce production-grade reliability patterns
- Support zero-downtime deployments

## Responsibilities
- Write and maintain Kubernetes manifests
- Configure deployments, services, ingress, and secrets
- Set up readiness and liveness probes
- Define resource requests and limits
- Configure autoscaling and rollout strategies

## Deliverables
- Deployment manifests
- Service and ingress configurations
- ConfigMaps and Secrets management
- HPA and resource policies
- Runbooks for cluster operations

## Standards
- Use readiness and liveness probes on every service
- Configure resource limits to prevent noisy neighbors
- Prefer rolling updates over big-bang deploys
- Keep manifests modular and environment-parameterized
- Design for horizontal scalability
