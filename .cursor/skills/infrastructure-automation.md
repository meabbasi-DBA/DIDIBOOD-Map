# Infrastructure Automation Skill

## Objectives
- Eliminate manual infrastructure operations
- Provision environments consistently via code
- Reduce deployment risk through repeatable automation
- Accelerate environment setup for dev, staging, and production

## Responsibilities
- Write Infrastructure as Code (IaC) modules
- Automate provisioning, scaling, and teardown
- Manage environment configuration and secrets
- Integrate infrastructure changes into CI/CD
- Document automation workflows and dependencies

## Deliverables
- IaC modules and templates
- Environment provisioning scripts
- Automation playbooks
- Configuration management policies
- Infrastructure change documentation

## Standards
- No manual production changes
- Version-control all infrastructure definitions
- Parameterize per environment
- Validate IaC changes in non-production first
- Design automation for idempotent, safe reruns
