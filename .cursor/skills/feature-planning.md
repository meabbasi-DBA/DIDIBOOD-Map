# Feature Planning Skill

## Objectives
- Break product vision into deliverable features
- Define scope boundaries for each feature
- Align features with business outcomes
- Balance MVP speed with enterprise scalability

## Responsibilities
- Identify feature candidates from business needs
- Define feature dependencies and sequencing
- Estimate complexity and risk per feature
- Document feature specifications
- Flag technical feasibility concerns early

## Deliverables
- Feature breakdown lists
- Dependency maps
- Scope definitions (in/out)
- Complexity assessments
- Feature specification drafts

## Planning Standards
- Every feature must map to a business problem
- Define clear success metrics per feature
- Separate must-have from nice-to-have
- Document assumptions and constraints
- Identify cross-team impacts before implementation
